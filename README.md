## 專案簡介

本專案將探討戰爭槍戰遊戲的開發過程，以及遊戲如何透過遊戲性、敘事、美術和技術元素來吸引玩家並提供極具娛樂價值的體驗。我們將研究成功的遊戲案例，以了解其設計和發展的最佳實踐。

## 預期成果指標

### 第一學期

期中KPI:
1. 研究Unity
2. 研究多人連線
3. 研究戰爭素材
4. 蒐集所需模型之資源
5. 完成基礎介面操作

期末KPI:
1. 新增前後動畫解說
2. 新增場景3D模型建置
3. 新增教官導航新手
4. 新增多人連線測試
5. 完成基礎遊戲機制

### 第二學期

期中KPI:
1. 體驗優化
2. 遊戲介面優化
3. 多人連線優化
4. 模型素材優化
5. 收集使用者意見回饋

期末KPI:
1. 完成遊戲軟體
2. 完成軟體實際測試
3. 與另一組結合VR
4. 針對回饋意見進行修改
5. 撰寫成果報告書
